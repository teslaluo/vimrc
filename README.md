# vimrc
我的个人vimrc，用着很顺手

vimrc: 就是.vimrc文件，一般放到~/.vimrc中
plugins.zip: vimrc文件中配置的三个插件(NERDTree/taglist/BufExplorer)打包，创建~/.vim目录，在~/.vim目录下执行unzip plugins.zip即可。

特性：

1、支持识别多种编码，并统一转成utf8输出

2、支持自动缩进

3、支持c++的代码通过空格折叠代码

4、支持方便的ctags跳转快捷键

5、支持NERDTree插件快捷键

6、支持taglist插件快捷键

7、支持BufExplorer插件快捷键

8、支持记住上次打开的位置
